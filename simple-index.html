App Features

Dashboard

Central hub with quick access to all key features including Mark Attendance, View Attendance, My Classes, Notifications, and Profile.

Mark Attendance

Quickly mark students as present, absent, or late with a simple tap interface. Shows class roster with student details.

View Attendance

View historical attendance records by date with statistics on present, absent, and late students.

My Classes

Browse all classes you teach with details like section, subject, and number of students.

Notifications

Receive updates about attendance reminders, new students, and administrative announcements.

Profile

View and edit your profile information, see teaching statistics, and manage app settings.

Backend Implementation

MongoDB Integration

  • Complete database models for Users, Classes, Students, Attendance records, and Notifications
  • Automatic fallback to in-memory storage when MongoDB connection isn't available
  • Secure authentication system with password hashing

API Endpoints

  • /api/user - User authentication and profile management
  • /api/classes - Teacher's class management
  • /api/classes/:classId/students - Student roster management
  • /api/classes/:classId/attendance - View attendance records
  • /api/attendance - Mark and update attendance
  • /api/notifications - Manage teacher notifications
