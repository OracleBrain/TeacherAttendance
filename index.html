Teacher Attendance Management Mobile App

This is a comprehensive Flutter-based mobile application designed specifically for teachers to manage classroom attendance, track student performance, and streamline administrative tasks. The app provides an intuitive interface for teachers to mark and view attendance, manage classes, receive notifications, and maintain their profile information.

Key Features

  • User authentication with secure login and registration
  • Interactive dashboard with quick access to all features
  • Mark attendance with Present, Absent, and Late options
  • View attendance records with detailed statistics
  • Manage classes and view student information
  • Notification system for important updates
  • User profile management
  • Mobile responsive design for seamless usage on any device

App Screens

Login Screen

Secure authentication page with:

  • Username and password fields
  • Remember me option
  • Registration option for new users
  • Informative onboarding section
Dashboard Screen

Central hub of the application with:

  • Colorful interactive cards for navigation
  • Quick access to all main features
  • Recent activities section
  • Notification indicators
Mark Attendance Screen

Efficient attendance recording with:

  • Class selection dropdown
  • Date picker
  • Student list with Present/Absent/Late options
  • Submit button to save attendance
View Attendance Screen

Comprehensive attendance records with:

  • Class and month selection
  • Attendance summary statistics
  • Detailed daily attendance records
  • Visual progress bars for attendance status
My Classes Screen

Class management interface with:

  • List of assigned classes
  • Class details including subject, section, and schedules
  • Student count information
  • Quick actions to view details or mark attendance
Notifications Screen

Alert and notification system with:

  • Read/unread indicators
  • Timestamp information
  • Notification categories with icons
  • Swipe actions to dismiss or mark as read
Profile Screen

User information management with:

  • Personal information display and editing
  • Account information section
  • Password change functionality
  • Logout option
